UE5 Digital Twin Framework

UCFPlugin 数字孪生城市敏捷开发

Universal CIM Framework — 基于 Unreal Engine 的数字孪生城市敏捷开发插件。支持轻量级蓝图项目、前后端协同的像素流项目，提供视口控制、漫游管理、输入配置等完备接口。

快速开始 查看 API 文档
快速开始
01

安装插件

将 UCFPlugin 文件夹放到项目的 Plugins 目录下，并为项目重新编译生成 C++ 工程

02

插件设置

在编辑器中打开 Project Settings > Plugins > UCFPlugin 配置以下选项：

配置项
BP Interface Path：添加蓝图接口所在的目录路径
Double Tap Time Interval：输入双击间隔时间（默认 0.23 秒）
Default Input Component Class：选择 UCFInputComponent
03

配置 GamePlay

选定主关卡，转到 World Settings 窗口，GameMode Override 设置为 UCFGameMode。Pawn 与 PlayerController 会自动匹配为对应的 Class。

此时可直接运行项目。若需自定义 Pawn 操作配置，可在场景中默认视点处拖入 UCFDefaultPawn 的实例，在 Details 中自定义配置。
04

接口扩展

在 AI Code IDE 中配置 skills 后，支持 AI 辅助 C++ 接口的扩展、更新、重构，并快速创建对应接口文档。

Pawn 控制方式 6 种操作

DefaultPawn 的基本控制以锚点为参考。鼠标按住时场景中出现绿色锚点标识，所有移动和旋转均基于该锚点计算。

行为描述默认控制方式
定点移动水平拖动控制 Pawn 左右移动到锚点位置；上下拖动控制 Pawn 前后移动到锚点位置按住 鼠标左键 拖动
定点旋转水平拖动以锚点为中心左右旋转；上下拖动以锚点为中心上下旋转按住 鼠标右键 拖动
定点缩放向前滚动逼近锚点；向后滚动远离锚点鼠标中键 滚动
快速聚焦快速聚焦到锚点位置鼠标右键 双击
竖向移动竖直向上 / 向下移动按住 Q / E 键
居中旋转以自身为中心左右 / 上下旋转按住 鼠标中键 拖动
车行模式 8 种操作
行为控制方式描述
前进按住 W 键车辆前进
后退按住 S 键车辆后退
左转按住 A 键车辆左转向，需同时前进/后退才移动
右转按住 D 键车辆右转向，需同时前进/后退才移动
制动单击 空格 键车辆制动（默认缓冲时间 1.5s），同时按 W/S 时制动失效
恢复单击 空格 键车辆侧翻时恢复正常姿态
切换视角模式单击 F 键跟随视角 vs 自由视角，默认为跟随视角
切换人称单击 V 键第一人称（车内沉浸）vs 第三人称（车外俯视），默认第三人称
若仅需要车行效果，请使用 /WebCore/GamePlay/Vehicle/BP_FakeVehicle.BP_FakeVehicle_C，调用 Turn 接口控制车行效果开关。
人行模式 5 种操作
行为控制方式描述
前进按住 W 键人物前进
后退按住 S 键人物后退
左转按住 A 键人物左转后前进
右转按住 D 键人物右转后前进
跳跃单击 空格 键人物跳跃障碍物
切换人称单击 V 键第一人称（沉浸）vs 第三人称（俯视），默认第三人称
转动视角鼠标 移动自由环顾四周
若仅需要人行效果，请调用 Turn 接口控制人行效果开关。
