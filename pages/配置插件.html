快速开始

1

安装插件

将 UCFPlugin 文件夹放到项目的 Plugins 目录下，并为项目重新编译生成 C++ 工程。

确保项目已配置为 C++ 项目。如果是蓝图项目，请先通过"添加 C++ 类"将其转换为 C++ 项目。
2

插件设置

在编辑器中打开 Project Settings > Plugins > UCFPlugin，配置以下选项：

  • BP Interface Path：添加蓝图接口所在的目录路径
  • Double Tap Time Interval：输入双击间隔时间（默认 0.23 秒）
  • Default Input Component Class：选择 UCFInputComponent
3

配置 GamePlay

选定主关卡，然后转到 World Settings 窗口，GameMode Override 设置为 UCFGameMode，Pawn 与 PlayerController 会默认匹配为对应的 Class。

此时可以直接运行项目，项目默认视点为启动时编辑器视点的位置或者 PlayerStart 实例的位置。

若需要自定义 Pawn 操作配置，可以在场景中自定义位置放置 UCFDefaultPawn 的实例，然后在 Details 中自定义配置。

4

控制 Pawn

UCFDefaultPawn 的基本控制都是以锚点为参考的，请始终注意鼠标按住时在场景中出现的绿色锚点，默认的控制方式如下：

定点移动

定点移动鼠标左键 + 拖动

按住鼠标左键水平拖动，控制 Pawn 左右移动到锚点位置；按住鼠标左键上下拖动，控制 Pawn 前后移动到锚点位置

定点旋转

定点旋转鼠标右键 + 拖动

按住鼠标右键水平拖动，控制 Pawn 以锚点为中心左右旋转；按住鼠标右键上下拖动，控制 Pawn 以锚点为中心上下旋转

定点缩放

定点缩放鼠标中键 滚动

向前滚动鼠标中键，控制 Pawn 逼近锚点；向后滚动鼠标中键，控制 Pawn 远离锚点

快速聚焦

快速聚焦双击鼠标右键

双击鼠标右键快速聚焦到锚点位置

竖向移动

竖向移动Q 上升 E 下降

按住 Q 键竖直向上移动；按住 E 键竖直向下移动

居中旋转

居中旋转鼠标右键 + 拖动（无锚点）

按住鼠标右键水平拖动，控制 Pawn 以自身为中心左右旋转；按住鼠标右键上下拖动，控制 Pawn 以自身为中心上下旋转
